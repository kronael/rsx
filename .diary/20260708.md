mislabeled and unfixable without amend/rebase (forbidden). Rule: **never
`git commit` while a code sub commits on the same tree**; if unavoidable, use
an explicit-pathspec commit (`git commit <file>`, only-mode) so it can't sweep
the index. Holding BUGS.md (fable + codex findings) + the publish roadmap
